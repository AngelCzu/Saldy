# Accessibility Guidelines

## Contrast Rules (WCAG 2.1)

### Text
- Normal text: ≥ 4.5:1
- Large text: ≥ 3:1

### Icons & UI
- Icons and interactive elements: ≥ 3:1

## Validated Combinations
| Foreground | Background | Usage |
|-----------|------------|------|
| #111827 | #F9FAFB | Main text |
| #6B7280 | #F9FAFB | Secondary text |
| #0D9488 | #F9FAFB | Links / icons |
| #FFFFFF | #0D9488 | Header titles |
